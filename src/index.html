关于我

我是 hoosin ，目前定居在广州，职业是一名程序员，混迹于互联网这个圈子差不多五年了。工作之外喜欢足球，我支持的主队是 广州恒大淘宝足球俱乐部 和 拜仁慕尼黑足球俱乐部 ，这么多年过去了，虽然 中国国家男子足球队 一直烂泥扶不上墙，但是我还是一如既然的支持着。毕业后先后在盛大网络和太平洋网络写代码，后来因为兴趣爱好出去创业过一次，搞了一个叫『嗨足球』的产品。

转眼到了 2015 年，我加入唯品会 多移动端创新团队，负责新技术研究。后来由于公司战略发展离开创新团队，目前就职于 唯品金融支付团队 。

写代码是一件美好的事情，我个人技术上侧重于 JavaScript/Node.js ，当然无聊的时候我也会写写 Golang 、 Python 等 , 喜欢 Node.js 全栈式 开发，对 前端架构 、前端构建工具、单元测试、自动化有比较丰富的经验。

生活中，我喜欢和一些好玩儿的人和团队，交流技术，谈谈人生。假如你觉得我有趣，我们可以聊聊啊。
